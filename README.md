# Property Insights Dashboard

A comprehensive property analysis tool that helps real estate professionals, investors, and property owners understand market trends and compare property values in their area.

## What is This?

The Property Insights Dashboard provides **Comparative Market Analysis (CMA)** reports for Australian properties. A CMA is a detailed report that helps determine a property's value by comparing it to similar properties that have recently sold in the same area.

## Why Use This Dashboard?

### For Real Estate Professionals
- **Quick Property Valuations**: Instantly generate CMA reports for any property
- **Client Presentations**: Share professional, visual reports with clients
- **Market Research**: Understand local market trends and property values
- **Competitive Analysis**: Compare multiple properties side-by-side

### For Property Investors
- **Investment Decisions**: Make informed decisions about property purchases
- **Market Trends**: Track price movements and growth rates over time
- **Area Analysis**: Understand which suburbs and regions are performing well
- **Comparable Sales**: See what similar properties sold for recently

### For Property Owners
- **Property Valuation**: Get an estimate of your property's current market value
- **Market Position**: Understand how your property compares to others in the area
- **Growth Tracking**: See how property values have changed over time

## How to Use

### Getting Started

1. **Enter a Property ID**: Type a CMA property ID in the search bar (e.g., `GANSW704079886`)
2. **View Results**: The dashboard automatically loads and displays:
   - Market summary cards at the top
   - Price trend chart on the left
   - Detailed comparison table on the right
3. **Explore Data**: 
   - Hover over chart points to see property details
   - Filter the table to find specific properties
   - Sort columns to organize data your way

### Understanding the Data

**Market Summary Cards** show:
- Where the property is located
- The median price for similar properties
- How much prices have grown (or declined)
- How many comparable properties were found

**Price Trend Chart** displays:
- Each point represents a property sale
- X-axis shows the sale date
- Y-axis shows the sale price in Australian Dollars
- Points are connected chronologically to show trends

**Comparison Table** lists:
- All similar properties that have sold recently
- Detailed information about each property
- Sortable and filterable for easy analysis

## Use Cases

### Scenario 1: Listing a Property
A real estate agent needs to determine the listing price for a new property. They enter the property ID, review the median price and growth rate, examine comparable sales, and use this information to recommend an appropriate listing price to the seller.

### Scenario 2: Making an Offer
A buyer is interested in a property but wants to ensure they're not overpaying. They search for the property, review recent comparable sales, check price trends, and use this data to make an informed offer.

### Scenario 3: Investment Analysis
An investor is evaluating multiple properties in different suburbs. They compare CMA reports for each property, analyze growth rates, review comparable sales, and identify the best investment opportunity.

### Scenario 4: Market Research
A property owner wants to understand how their property value has changed. They search for their property, review the growth rate, examine recent comparable sales, and see how their property compares to others in the area.
